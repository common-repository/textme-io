This plugin adds TextMe.io SMS functionality to your site. 

== Description ==

The official TextMe.io Wordpress plugin gives you a quick shortcode that embeds the form in any post or page. 



== Installation ==
1.  Sign up for TextMe.io
2.  Grab your User ID in the dashboard. 
3. Upload the plugin. 
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Paste your TextMe.io User ID into the TextMe Plugin Settings Page
6.  Paste the shortcode [textme] wherever you want the form to appear. Any post or page or widget or anything!
